# Video Placeholder

Place your hero video here as: `hero.mp4`

## Requirements
- Format: MP4 (H.264)
- Resolution: 1920×1080 or higher
- Duration: 15–60 seconds (loopable)
- File size: Keep under 15 MB for fast loading

## Mobile behaviour
On screens < 768px the video is hidden; a CSS gradient is shown instead — no video loaded on mobile.

## Fallback
If video fails: `public/images/hero-poster.jpg` is shown via the `poster` attribute.
